Project: Radar-Mesh

Real-time 3D Human Pose and Mesh Reconstruction using Phaser Radar Micro-Doppler Signatures

This project demonstrates a novel approach to reconstruct a real-time, 3D human mesh using only micro-Doppler signatures from the Phaser FMCW radar kit. This privacy-preserving, low-cost, and robust system translates complex radar signals into an intuitive 3D visualization of human motion, with vast applications in security, healthcare, and human-computer interaction.
